# 2025-11-13 — Внешний тест лимитов сообщений (RATE_LIMIT_KV)

- **Источник:** подтверждение заказчика (Telegram стенд).
- **Текущие параметры:** лимит по умолчанию (20 сообщений в сутки), окно ~24 часа.
- **Что проверено:**
  - Пока лимит не достигнут — бот продолжает диалог без ограничений.
  - После достижения лимита бот перестаёт отвечать и отправляет текст: «Дневной лимит ответов исчерпан. Попробуйте снова через 13 ч 57 мин.»
- **Что ещё не проверялось:** работа таймера обратного отсчёта, автоматический сброс лимита по окончанию окна.
- **Пожелание заказчика:** на период тестов поднять дневной лимит и сократить окно ожидания, чтобы быстрее циклить проверки.
- **Следующие шаги:** оформить отдельный шаг дорожной карты «Настройка и проверка лимитов сообщений на пользователя», описать изменение параметров и повторно задокументировать проверку с новым порогом/окном.
